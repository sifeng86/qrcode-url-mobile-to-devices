FAQ

Frequently asked questions

Do I need to sign up or install an app?

No. Sendline works in the browser on both devices and does not require sign-up or login.

What can I send?

You can send links and short notes immediately. File sending works when the server has temporary cloud storage configured.

Do file shares expire?

Yes. File shares use a configurable lifetime, and the object is deleted after that lifetime expires.
